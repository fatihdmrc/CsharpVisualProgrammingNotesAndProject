# C# Görsel Programlama Notları 

Bu repo, Bandırma Onyedi Eylül Üniversitesi Yönetim Bilişim Sistemleri bölümü Görsel Programlama dersi boyunca oluşturulan projeleri düzenli bir yapıda toplar.

## Klasör Yapısı
- `AnalogSaat/` 
- `Eşleştirme Oyunu/`
- `Hesap Makinesi/`
- `Kronometre/` 
- `Not Defteri/` 

## İçerik

- Her klasör bir projeden oluşur.

## Çalıştırma

1. `Görsel Programlama.sln` dosyasını Visual Studio ile açın.
2. İstediğiniz projenin csproj dosyasını seçin.
3. Projeyi `Build` veya `Run` ile çalıştırın.
